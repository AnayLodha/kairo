# kairo

A calm, student-friendly space to manage academics, track mood, capture creative ideas, and build habits that last.

## Features

- **Academics**: Track your subjects and marks.
- **Mood Check-in**: Daily mood tracking and insights.
- **Creative Zone**: Capture your ideas and thoughts.
- **Daily Focus**: Manage your daily tasks.
- **Habit Tracking**: Build streaks and monitor your progress.

## Tech Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   cd ikigai-atlas
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
